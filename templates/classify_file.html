This application is a demo for prioritizing customer support requests based on sentiment and urgency of the message.
You can input a string or file of strings (matching the requirements below) to see how they'd be classified.


Classify a String

Submit an example of a customer support ticket in any given language to be classified.




Classify a File of Tickets

Submit a .csv file with no header, and the text for each ticket in the first column with no other data.
For testing, you can download and use the sample_input.csv from the GitHub repo in the flask_upload directory.
